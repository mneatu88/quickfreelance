# Contributing to QuickFreelance

Thank you for considering contributing to QuickFreelance!

---

## How to Contribute

1. **Fork** the repository
2. Create a new branch (`git checkout -b feature/amazing-feature`)
3. Make your changes
4. Commit your changes (`git commit -m 'Add amazing feature'`)
5. Push to the branch (`git push origin feature/amazing-feature`)
6. Open a **Pull Request**

---

## Development Setup

- The project is built with **Lovable.dev + Supabase**
- Main documentation is in the `/docs` folder
- See `docs/architecture.md` for technical overview
- See `docs/roadmap.md` for current priorities

---

## What We Need Help With

- Bug reports and feature requests
- UI/UX improvements
- Documentation improvements
- Testing new features
- Marketing & social media content

---

## Code of Conduct

- Be respectful and professional
- Keep discussions constructive
- Focus on making the platform better for freelancers

---

**Questions?**  
Open an Issue or contact us at support@quickfreelance.co.uk
